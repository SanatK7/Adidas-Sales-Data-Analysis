# Adidas Sales Data Analysis

This Python script utilises data from Adidas sales to perform analysis and Tableau to generate insights. The analysis includes fetching sales data, performing exploratory data analysis (EDA), and creating visualisations such as sales trends and profitability metrics.

## Prerequisites

Before running the script, ensure you have the required Python libraries installed. You can install them using the following:
Numpy
Pandas
Matplotlib
Seaborn
Plotly
Datetime



Visualisations


1. Sales by Month

The line graph illustrates the monthly sales trends, providing insights into the fluctuation and seasonality of Adidas sales over time.

2. Sales by State

A choropleth map visually represents Adidas sales across different states, offering a geographical perspective on regional sales performance.
3. Sales by Sales Method
  
The pie chart showcases the distribution of Adidas sales based on various sales methods, aiding in understanding the effectiveness of different sales channels.
4. Sales by Retailer 

The bar chart provides a comparative analysis of sales figures across different retailers, offering a clear visualisation of the top-performing retail partners.


These visualisations offer a comprehensive overview of Adidas sales data, enabling stakeholders to make informed decisions and identify key trends in sales performance.


Recommendations

The script provides recommendations based on sales data analysis, aiding in strategic decision-making.
